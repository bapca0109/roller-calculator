**original_problem_statement**: The user wants to create a mobile application to calculate the price of belt conveyor rollers, functioning as a comprehensive engineering and quoting tool. The project has evolved to include complex pricing logic, freight, value-based discounts, GST, a multi-product quote builder, PDF exports, and company branding.

**PRODUCT REQUIREMENTS**:
- **Core Function**: Calculate prices for Standard (Carrying) and Impact belt conveyor rollers based on detailed specifications (Pipe dia/length/thickness, Shaft dia, Bearing type/make, etc.).
- **Pricing Formula**: .
- **Value-based Discounts**: Apply a tiered discount (5% to 35%) on the total order value before packing and taxes.
- **Additional Charges**:
    - **Packing**: Optional charges (Standard: 1%, Pallet: 4%, Wooden Box: 8%).
    - **Freight**: Calculated per kg based on distance from dispatch pincode (382433), determined via an external API.
- **GST Calculation**: Apply 18% GST.
    - **CGST (9%) + SGST (9%)**: For destinations within the same state (Gujarat).
    - **IGST (18%)**: For destinations in other states.
- **Quoting System**:
    - Users must be able to add multiple, different roller configurations to a single quote.
    - Save quotes and view them in a dedicated Quotes tab.
    - Export saved quotes to a professional-looking PDF document.
- **UI/UX**:
    - The app should feature the company's logo (CONVERO SOLUTIONS).
    - Input fields must have real-time validation with clear inline error messages.
- **Data Management**:
    - The user must be able to export all raw material pricing data to an Excel spreadsheet for review.

**User's preferred language**: English

**what currently exists?**
The project is a feature-complete Expo (React Native) and FastAPI application.
- **Backend**: Handles all business logic including roller cost calculation, multi-tier discounts, packing charges, distance-based freight, and state-based GST (CGST/SGST/IGST). It serves an endpoint to download an Excel file of all raw material prices. All pricing data is hardcoded in .
- **Frontend**: A fully functional mobile UI built with Expo Router. It includes:
    - A detailed calculator form with real-time validation and dropdowns populated from the backend.
    - A results section displaying a full cost breakdown including discounts, packing, GST, and freight.
    - A Quote Builder for creating multi-product quotes.
    - A Quotes tab to list all saved quotes and view their details in a modal.
    - A PDF export feature for quotes.
    - The company logo is present on the login and calculator screens.
