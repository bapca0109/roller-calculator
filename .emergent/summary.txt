**original_problem_statement**: The user wants to create a mobile application to calculate the price of belt conveyor rollers. The app should function as a comprehensive engineering product and quoting tool.

**PRODUCT REQUIREMENTS**:
- **Product Type**: The application is exclusively for calculating the price of belt conveyor rollers.
- **Product Configuration**:
    - Users must be able to select roller specifications, including:
        - Roller Type: Standard (Carrying) or Impact.
        - Pipe Diameter (as per IS-9295).
        - Pipe Thickness (Type A/B/C as per IS-1239).
        - Pipe Length (as per IS-8598).
        - Shaft Diameter.
        - Bearing Type (e.g., 62xx, 63xx, 4202xx series) and Make (China, SKF, FAG, Timken).
        - For Impact Rollers: Rubber Diameter.
- **Costing Formula**: The final price is calculated as: .
- **Raw Materials**: The total raw material cost includes:
    1.  Pipe (cost/kg, weight based on OD and thickness).
    2.  Shaft (cost/kg).
    3.  Bearings (price per piece, based on type and make).
    4.  Housing (price per piece, based on pipe OD and bearing OD).
    5.  Seals (price per set, based on bearing type).
    6.  Circlips (4 pcs per roller, price based on shaft dia).
    7.  Rubber Rings (for Impact rollers, cost based on pipe/rubber OD and length).
    8.  Locking Rings (for Impact rollers, fixed cost per roller based on pipe OD).
- **Product Code Generation**: The app must automatically generate a product code (SKU) based on the selected specifications (e.g., ).
- **Additional Charges**:
    - **Packing**: Optional charges (Standard: 1%, Pallet: 4%, Wooden Box: 8%) applied after the main price calculation.
    - **Freight**: Optional charge calculated per kg based on the distance from a fixed dispatch pincode (382433) to the destination pincode.
- **User Roles**: The system should support multiple user roles (Admin, Sales, Customer) with different permissions for viewing quotes and managing products.
